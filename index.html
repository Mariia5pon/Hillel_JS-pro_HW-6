6.1. Написати функцію видалення масиву символів. 6.2. Визначення середнього арифметичного. 6.3. Функція видалення елементу

6.1. Написати функцію видалення масиву символів.

Створити функцію, яка прибирає з рядка всі символи, які ми передали другим аргументом. 'func(" hello world", ['l', 'd'])' поверне нам "heo wor". Вихідний рядок та символи для видалення задає користувач.

6.2. Визначення середнього арифметичного.

Дано масив з елементами різних типів: [1, 'hello', 3, true, 5, '564899', null, 2.5]. Створити функцію, яка вираховує середнє арифметичне лише числових елементів даного масиву.

6.3. Функція видалення елементу

Реалізуйте функцію removeElement(array, item), щоб видалити елемент item з масиву array.

Наприклад:

const array = [1, 3, 4, 6, 2, 5, 7];
removeElement(array,4);
console.log(array); // Результат: [1, 3, 6, 2, 5, 7]
